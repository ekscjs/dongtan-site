# 작업지시서 — 카드/섹션 상단 작은 라벨 글씨 키우기 (2026-07-16)

> 배경: `text-xs font-semibold text-[#9B4DAB]` 조합(카드·섹션 위에 붙는 짧은 라벨 — "몸 상태 체크",
> "1분 셀프체크", "통증지도", "미소AI", "바디 리셋 세션" 등)이 site 전체 14곳에서 12px로 너무
> 작음. 홈 히어로 라벨("동탄 기능개선·재활·체형교정 전문")은 이미 14px(`text-sm`)를 쓰고 있어서
> 이 참에 사이트 전체를 그 크기로 통일. 시니어·중장년층 타겟 고려.

## 수정 방법
`src/` 전체에서 **정확히 이 문자열**을 찾아 바꾼다 (전부 동일 패턴이라 일괄 치환 가능,
`text-xs`만 있고 뒤에 `font-semibold text-[#9B4DAB]`가 안 붙은 다른 곳은 절대 건드리지 말 것):

- 찾기: `text-xs font-semibold text-[#9B4DAB]`
- 바꾸기: `text-sm font-semibold text-[#9B4DAB]`

해당되는 파일 8개, 총 14곳(확인 완료 — 전부 이 교체 대상):
- `src/app/ask/AskConcierge.tsx` (1곳)
- `src/app/check/CheckQuiz.tsx` (5곳)
- `src/app/check/pain/PainMap.tsx` (4곳)
- `src/app/page.tsx` (1곳)
- `src/app/programs/page.tsx` (1곳)

(정확한 라인 번호는 코드 수정 중 바뀔 수 있으니 문자열 검색으로 찾아서 전부 교체할 것 — 위 8개
파일에서 `text-xs font-semibold text-[#9B4DAB]` 문자열이 나오는 곳은 전부 대상.)

## 하지 말 것
- `text-xs`만 있고 `font-semibold text-[#9B4DAB]`가 같이 안 붙은 다른 곳(예: 안내문구, 날짜,
  버튼 보조텍스트 등)은 절대 건드리지 말 것 — 이번 작업은 저 세 클래스가 정확히 함께 붙은
  "상단 작은 라벨" 용도에만 해당.

## 검증
1. `/check`, `/check/pain`, `/ask`, 홈, `/programs` 페이지 돌아보면서 상단 작은 라벨들이 전부
   한 단계 커졌는지(레이아웃 깨짐 없는지도 같이 확인).
2. `npm run build` 통과.
