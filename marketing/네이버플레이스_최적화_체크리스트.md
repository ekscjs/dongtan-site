# 네이버 플레이스 최적화 체크리스트 (동탄 단기 유입용)

> 한국 지역밀착 업종은 네이버 검색·지도가 메인 유입로입니다. 플레이스 상위 노출 = 예약·문의로 직결.
> 우리 플레이스: https://map.naver.com/p/entry/place/1101035370

---

## 1. 기본 정보 — 정확하고 "일관되게"

- [ ] 상호: **내몸에미소** (사이트·구글·카톡과 100% 동일하게)
- [ ] 주소: 경기도 화성시 동탄 지성로 134 5층 (한 글자도 다르지 않게)
- [ ] 전화 / 카카오톡 채널 연결
- [ ] 영업시간: 평일 10:00–22:00 / 주말·공휴일 휴무
- [ ] 주차: 건물 내 주차 가능
- [ ] 세부 업종: 재활·체형교정·기능개선 운동에 가장 맞는 업종으로 정확히 설정

> ⚠️ **NAP(상호·주소·전화) 일관성**이 핵심. 사이트·구글·네이버·카톡이 전부 같아야 네이버·구글 둘 다 신뢰도가 올라갑니다.

## 2. 대표키워드 & 소개글

- [ ] 플레이스 키워드에 **동탄 재활 · 동탄 체형교정 · 동탄 통증 · 동탄 운동센터** 등 타겟 키워드 반영
- [ ] 업체 소개글에 지역+서비스 키워드를 자연스럽게, 차별점 명시 ("증상이 아닌 원인부터", "1:1", 공식 인증, 누적 166명·재등록률 80%)

---

## 3. 사진 (네이버는 사진 수·신선도 중요)

- [ ] 외관·간판·입구 (길찾기에 도움)
- [ ] 내부 전경 · 수업 장면 · 운동 도구
- [ ] 비포애프터 (고객 동의 필수)
- [ ] 원장/지도자 + 공식 인증서
- [ ] **정기 업데이트** (새 사진 꾸준히 = 활성 신호)

---

## 4. 리뷰 (방문자 + 영수증 + 블로그)

- [ ] **영수증 리뷰** 유도 — 신뢰도 가장 높음. 결제 후 안내
- [ ] **방문자 리뷰** — 위 '리뷰 수집 키트' 시스템으로 요청
- [ ] **블로그 리뷰** — 회원·지인의 진짜 방문기 + 우리 네이버 블로그에도 후기성 글
- [ ] 모든 리뷰에 답글 (지역·서비스 키워드 한 번씩 자연스럽게)

---

## 5. 예약 · 톡톡 · 스마트콜

- [ ] **네이버 예약 연동** — 체험/상담 예약 받기. 전환율 + 노출 가산점
- [ ] **네이버 톡톡** 켜고 응답률 관리 (빠른 응답 = 노출 유리)

---

## 6. 소식 · 활성도

- [ ] 정기적으로 소식/이벤트 업로드 (새 글, 프로그램, 변화 사례)
- [ ] 꾸준한 활동 = 네이버가 "살아있는 업체"로 인식

## 네이버 vs 구글 역할 정리

| | 네이버 플레이스 | 구글 (사이트+GBP) |
|---|---|---|
| 역할 | **단기 유입** (지금 예약) | **장기 블루오션** (1년 선점) |
| 핵심 | 리뷰·사진·예약·키워드 | 콘텐츠·구조화데이터·리뷰 |
| 경쟁 | 치열함 | 동탄 로컬은 아직 여유 |

→ 둘 다 가되, **리뷰는 받을 때 양쪽 동시에** 유도하면 효율 최대.
