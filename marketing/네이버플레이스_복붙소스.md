## ① 대표키워드 (네이버 스마트플레이스 → 정보 → 대표키워드)

대표키워드 칸에 우선순위대로 등록 (보통 5개 내외):

```
동탄재활  동탄체형교정  동탄PT  동탄통증운동  동탄운동센터
```

여유 있으면 추가로 노릴 키워드 (소개글·소식글에 자연스럽게 녹이기):

```
동탄 재활운동 · 동탄 자세교정 · 동탄 거북목 · 동탄 허리통증 · 동탄 골반교정 · 화성 재활 · 동탄 시니어 운동
```

## ② 업체 소개글 (정보 → 상세설명/소개)

```
동탄 내몸에미소는 기능개선·재활·체형교정·통증 완화 운동을 1:1로 진행하는 동탄 운동센터입니다.

허리·무릎·어깨가 불편한데 검사하면 "이상 없다"는 분, 치료받으면 잠깐 괜찮다가 다시 재발하는 분, 거북목·굽은 등·골반 틀어짐이 신경 쓰이는 분, 수술·부상 후 회복이 필요한 분까지 — 증상이 아닌 '움직임의 원인'을 찾아 해결합니다.

지도자 전원이 스포츠지도사 자격을 보유하고 있으며, 국민체육진흥공단 스포츠·장애인 바우처 가맹 시설, 국민체력100 협력 센터, 특허청 스포츠 교육기관 특허를 받았습니다. 누적 회원 166명, 재등록률 80%로 동탄에서 4년 넘게 함께해 왔습니다.

무료 상담 → 1회 체험 → 맞춤 프로그램 순으로 부담 없이 시작하실 수 있습니다. 동탄 지성로 134 5층, 평일 10:00–22:00 운영(건물 내 주차 가능). 카카오톡 채널 또는 네이버 예약으로 편하게 문의하세요.
```

## ③ 첫 소식글 (소식 → 새 소식)

제목:
```
스트레칭·마사지로도 안 풀리는 허리, 이유가 있습니다
```

본문:
```
스트레칭도 하고 마사지도 받았는데 허리 깊은 곳이 여전히 묵직한 분들, 생각보다 많으세요.

표면 근육과 심부 근육은 접근법이 다릅니다. 깊은 곳에 있는 근육은 힘으로 누른다고 풀리지 않아요. 오히려 더 수축합니다. 심부는 '힘'이 아니라 '호흡'으로 풀어야 합니다.

자세한 이야기는 블로그에 정리해두었어요 👇
https://dongtan.naemiso.com/blog/why-deep-back-pain-wont-go-away

동탄에서 통증·재활·체형교정, 증상이 아닌 원인부터 찾는 1:1 운동.
궁금한 점은 카카오톡 채널이나 네이버 예약으로 편하게 문의주세요. 😊
```

## 붙여넣고 나서 (사진·예약·톡톡)

- [ ] 사진: 외관/간판·내부·수업장면·도구·비포애프터(동의)·인증서 업로드
- [ ] 네이버 예약: '체험/상담 예약 받기' 연동 켜기
- [ ] 네이버 톡톡: 켜두고 빠른 응답
- [ ] 이후 소식글은 새 블로그 발행할 때마다 위 ③번 형식으로 1개씩 (블로그↔플레이스 교차 홍보)

> 팁: 소개글·소식글에 "동탄 + 서비스" 키워드가 자연스럽게 들어가 있어서, 따로 키워드를 욱여넣지 않아도 됩니다. 읽기 자연스러운 게 우선이에요.
